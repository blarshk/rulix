# Rulix

Rulix is a gem for defining and using rules to manipulate, validate, and transform datasets.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rulix'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rulix
